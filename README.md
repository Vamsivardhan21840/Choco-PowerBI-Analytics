# Choco Power BI Data Analytics Project

## Project Overview
This is an end-to-end **Power BI data analytics project** designed to analyze chocolate sales performance across products, regions, shipments, and sales teams.

The dashboard converts raw data into meaningful business insights, helping stakeholders understand revenue trends, profit margins, shipment distribution, and individual sales performance.

---

## Project Objectives
• Analyze overall sales amount and profit percentage  
• Track month-wise sales and box trends  
• Identify top-performing products and regions  
• Evaluate shipment distribution and efficiency  
• Compare sales members based on key performance metrics  

---

## Tools & Technologies Used
• Power BI  
• Microsoft Excel  
• CSV datasets  
• Data modeling and DAX  
• Data visualization  

---

## Key Insights from the Dashboard
• Total Sales Amount: 141.49M  
• Total Boxes Sold: 9M  
• Overall Profit Percentage: 57.3%  
• Total Shipments: 25K  
• Top revenue-generating regions: India, New Zealand, Australia  
• Product-level profit and performance comparison  
• Salesperson-wise performance ranking
